# Betnet SMS Sender v1.5

برنامه دریافت اعلان پیامک و ارسال آن به وب‌هوک.

## تغییرات نسخه 1.5
- کلید ارسال خودکار در صفحه نخست و ذخیره فوری وضعیت
- ادامه فعالیت Notification Listener بعد از بسته‌شدن رابط برنامه
- راه‌اندازی مجدد Listener پس از روشن‌شدن گوشی یا آپدیت برنامه
- دسترسی اجرای پس‌زمینه و حذف محدودیت باتری
- انتقال انتخاب برنامه پیامک به تنظیمات پیشرفته
- آیکون مینیمال سیاه‌وسفید جدید
- رابط فارسی راست‌چین و فونت سیستمی مناسب فارسی

> اندروید برای Auto-start رابط استاندارد مشترکی ندارد. دکمه «دسترسی اجرای خودکار و باتری» صفحه رسمی باتری گوشی را باز می‌کند. در گوشی‌های سامسونگ، برنامه را از Sleeping apps خارج کنید.

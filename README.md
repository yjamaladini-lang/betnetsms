# Betnet SMS Sender v1.4

نسخه اختصاصی ارسال اعلان پیامک به وب‌هوک.

## امکانات
- رابط کاملاً راست‌چین و فارسی با طراحی مینیمال سیاه و سفید
- داشبورد جداگانه برای برنامه‌های ارتباطی، وب‌هوک‌ها، تاریخچه، ارسال دستی و تنظیمات
- اجبار کاربر به فعال‌کردن Notification Access یا خروج از برنامه
- انتخاب برنامه‌های ارتباطی با نام، آیکون و نام پکیج
- فیلتر فرستنده و متن
- تلاش مجدد زمان‌بندی‌شده و توقف بعد از اولین موفقیت
- تاریخچه شمسی و جزئیات پاسخ هر تلاش
- ارسال دستی و ارسال مجدد
- اعلان موفق یا ناموفق ارسال

## ساخت APK
Workflow گیت‌هاب `Build Betnet APK` بعد از Push اجرا می‌شود و APK Debug را در Artifacts قرار می‌دهد.

تغییرات بخش تاریخچه پیام‌ها

- اضافه‌شدن دکمه بازگشت در صفحه تاریخچه
- اضافه‌شدن حالت انتخاب و حذف گروهی تاریخچه
- لمس طولانی روی پیام، حالت انتخاب را فعال می‌کند
- وضعیت سبز: پیام به سایت رسیده و matched=true بوده؛ بسته تحویل شده
- وضعیت زرد: پیام با HTTP موفق به سایت رسیده ولی matched=false بوده؛ بسته تحویل نشده
- وضعیت قرمز: پیام به سایت نرسیده یا پاسخ HTTP ناموفق بوده
- بازطراحی کارت‌های تاریخچه و پنجره جزئیات مطابق نمونه ارسالی
- نمایش پاسخ وب‌هوک، شماره تلاش و HTTP در جزئیات
- دکمه ارسال مجدد در جزئیات حفظ شده است

نکته: پروژه سورس است و باید با Android Studio یا GitHub Actions موجود در پروژه کامپایل شود.
